Mixed-methods research program investigating how autonomous AI agents can be manipulated,
subverted, and weaponized — and how to build systematic defenses.

**Methodology:**
- Delphi expert study with cybersecurity practitioners
- Scenario-based lab demonstration on Proxmox VE isolated infrastructure
- Cloud zones on Oracle Cloud and AWS ap-south-1 for remote Delphi panel

**Attack surface taxonomy being developed:**
- Prompt injection in multi-agent pipelines
- Goal hijacking via adversarial tool responses
- Memory poisoning across agent sessions
- Tool-call manipulation and exfiltration

**Lab stack:** Proxmox VE, Ubuntu VMs, Python, LangGraph, OpenRouter, local LLMs
